
# Setup
- Download latest version then activate using license file
- license file is in gmail, search for 'better touch tool'

# Import config
- top right 'Preset:default', then press 'Import' button
- 

### How to change default font?
https://community.folivora.ai/t/choose-default-font-for-touch-bar/14381/2
defaults write com.hegenberg.BetterTouchTool BTTTouchBarCustomFontName THENAMEOFTHEFONT
Example:
defaults write com.hegenberg.BetterTouchTool BTTTouchBarCustomFontName Iosevka
